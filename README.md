 HR-DATASET-ANALYSIS-VISUALIZATION
HR Dataset Analysis &amp; Visualization project using Power BI. This project focuses on employee attrition, department-wise performance, salary trends, gender diversity, and experience analysis. Interactive dashboards help HR teams gain data-driven insights for better decision-making.

Objectives

Analyze employee attrition patterns

Understand department and job role distribution

Study salary and experience impact

Visualize gender diversity

Create interactive HR dashboards

 Dataset

The dataset includes employee information such as age, gender, department, job role, income, experience, performance rating, and attrition status.

 Key Insights

Higher attrition in specific departments

Employees with lower experience and salary show higher attrition

Gender imbalance in certain job roles

Performance ratings influence employee retention

 Tools Used

Microsoft Power BI

Power Query

DAX

 How to Use

Download the .pbix file

Open it in Power BI Desktop

Explore dashboards using filters and slicers

 Conclusion

This Power BI project helps understand employee behavior and supports HR teams in improving retention and workforce planning.
